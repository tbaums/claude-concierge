# Theming

The look comes from two layers: the **iTerm2 profile** (font, window colors,
badge, blur) and the **tmux config** (status bar, pane borders, copy-mode
colors). They share one neon/synthwave palette.

## Palette

| Role | Hex |
|------|-----|
| Background | `#15101f` / `#1a1130` |
| Foreground | `#e6dcff` |
| Magenta (accent) | `#ff2e97` |
| Cyan | `#00e8ff` |
| Purple | `#b14bff` |
| Gold | `#ffd23f` |

iTerm colors are generated by `config/iterm-profile.py` (edit the `col(...)`
values there and re-run it, or re-run `install.sh`). tmux colors live in the
`NEON / SYNTHWAVE THEME` block of `config/tmux.conf`.

## Font

The installer uses **Monaspace Neon** (Nerd Font) at 15pt. Monaspace ships five
fun-but-computery families — swap by setting `CONCIERGE_FONT` before install, or
edit `"Normal Font"` in the generated profile:

| Family | PostScript name | Vibe |
|--------|-----------------|------|
| Neon (default) | `MonaspaceNeonNF-Regular` | clean, neony |
| Argon | `MonaspaceArgonNF-Regular` | humanist sans |
| Xenon | `MonaspaceXenonNF-Regular` | slab serif |
| Radon | `MonaspaceRadonNF-Regular` | handwriting |
| Krypton | `MonaspaceKryptonNF-Regular` | mechanical / sci-fi |

```sh
CONCIERGE_FONT="MonaspaceKryptonNF-Regular 15" bash install.sh
```

A Nerd Font is recommended so the status-bar powerline separators and glyphs
render. Any installed monospace font name works, though.

## The banner & badge

Two things keep the window unmistakable:

- a persistent **top status bar** reading `🤵‍♀️ CLAUDE CONCIERGE` with the
  session name, copy/paste hint, and a clock (in `tmux.conf`);
- a translucent **`CONCIERGE` badge** watermarked into the iTerm window itself
  (`Badge Text` in the profile).

Change the emoji/character in both `status-left` and `set-titles-string` in
`tmux.conf`, and `Custom Window Title` in the profile.
